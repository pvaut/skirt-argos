

# Sample data

The first time you open the ARGOS url in your browser, it **automatically fetches a couple of sample data sets**.
Once downloaded, you can start exploring these withough having to configure anything. Just click on a tile on the start page.

These data sets have not been chosen for their scientific importance, but rather to serve as a showcase of the capabilities of ARGOS.
They come with a pre-configured dashboard that illustrates how ARGOS can be tuned via its template mechanism. You can start familiarizing yourself with ARGOS's capabilities, and for example adapt the dashboard.

Note that **these sample data sets cannot be removed**: if you remove them, ARGOS will automatically re-fetch them the next time it is reloaded in the browser.
